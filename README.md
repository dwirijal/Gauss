# Gauss: Finance and Trading Agent

Agent keuangan & trading berbasis Python (Langchain & Langgraph) untuk dwizzyOS.

## Config
Konfigurasi API Key & Database di `.env`:
```env
OPENAI_API_KEY=your_key
# Atau custom provider (9Router)
OPENAI_API_BASE=https://api.9router.com/v1
```

## Run
```bash
uv run python -m gauss.main
```
